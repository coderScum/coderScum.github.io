Signing for “Unity-iPhone” requires a development team. Select a development team in the project editor.
Code signing is required for product type ‘Application’ in SDK ‘iOS 10.0’

2.arc机制机制相关

使用的 webview.mm中有错误。

2.1 禁用在Xcode中的特定文件的ARC

项目使用了arc机制而有些文件禁止使用而报错。
点击项目名，在中间一栏选择targets,然后选择build phases选项。
展开Compile Sources，找到你报错的文件名，然后双击添加-fno-objc-arc。

http://blog.csdn.net/wildcatlele/article/details/9997535

3.bitcode 相关

Libraries/Plugins/iOS/libEasyARUnity.a(libEasyARUnity.a-arm64-master.o)’ does not contain bitcode. You must rebuild it with bitcode enabled (Xcode setting ENABLE_BITCODE), obtain an updated library from the vendor, or disable bitcode for this target. for architecture arm64

禁用 bitcode。
