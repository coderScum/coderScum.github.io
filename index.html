AssetBundleManifest

打包最后会生成 manifest文件，每个ab文件对应一个，以及存储目录对应一个，存储目录对应的这个manifest文件记录了所有文件的里面记录了所有文件的依赖关系，而每个文件对应的manifest中记录了文件的hash。在做热更新的时候可以在打包的最后根据manifest生成一个json文件。Unity3d5.3以上已经支持json格式文件的解析。

为 Mac OS X 构建插件

在 Mac OSX 上，插件被部署为束。选择文件 (File)->新工程 (NewProject…)，然后选择束 (Bundle) -> Carbon/Cocoa 可加载束 (Carbon/Cocoa Loadable Bundle) (XCode 3) 或 OS X -> 框架和库 (Framework & Library) -> 束 (Bundle) (XCode 4)，即可创建带 XCode 的束工程。
